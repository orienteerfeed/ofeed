# Files Tab — Import State Metadata Display Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Show `EventImportState` metadata (creator, externalStatus, lastSuccessfulImport, successCount, skippedCount, click-to-copy rawHash) as small text inside each FileSection card in the Files settings tab, fetched via a new `eventImportStates` GraphQL query.

**Architecture:** A new standalone `eventImportStates(eventId)` GraphQL query is added alongside the existing `eventFilesStatus` query in `course.graphql.ts`. The client fetches both queries in parallel; import state rows are mapped by `payloadType` (StartList / CourseData / ResultList) to the matching card. A new inline `ImportStateMeta` component renders the metadata below the drag-drop upload zone.

**Tech Stack:** Pothos (server GraphQL), Apollo Client (client query), React, Tailwind 4, lucide-react, date-fns, shadcn Separator, react-i18next, Vitest + Testing Library.

- [ ] **Step 4: Add query, types, and mapping**

After the existing `UPDATE_CONTROL_RADIO_FLAG` gql block, add:

```ts
export const GET_EVENT_IMPORT_STATES = gql`
  query EventImportStates($eventId: String!) {
    eventImportStates(eventId: $eventId) {
      sourceType
      payloadType
      rawHash
      creator
      externalStatus
      lastSuccessfulImportAt
      successCount
      skippedCount
    }
  }
`;

export type ImportStateInfo = {
  sourceType: string;
  payloadType: string;
  rawHash: string;
  creator: string | null;
  externalStatus: string | null;
  lastSuccessfulImportAt: string | null;
  successCount: number;
  skippedCount: number;
};

type EventImportStatesData = {
  eventImportStates: ImportStateInfo[];
};

const PAYLOAD_TO_SECTION = {
  StartList: 'startList',
  CourseData: 'courses',
  ResultList: 'results',
} as const;

type SectionKey = keyof typeof PAYLOAD_TO_SECTION;
type SectionName = (typeof PAYLOAD_TO_SECTION)[SectionKey];
type ImportStateMap = Partial<Record<SectionName, ImportStateInfo>>;

function buildImportStateMap(states: ImportStateInfo[]): ImportStateMap {
  const map: ImportStateMap = {};
  for (const state of states) {
    if (state.sourceType !== 'IOF_XML') continue;
    const section = PAYLOAD_TO_SECTION[state.payloadType as SectionKey];
    if (section) map[section] = state;
  }
  return map;
}
```

const ImportStateMeta = ({
  t,
